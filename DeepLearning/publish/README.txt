=================================================================
  SoapDetector — Object Detection Application
=================================================================

QUICK START:
  Double-click "SoapDetector.bat" to run the application.

FOLDER STRUCTURE:
  app/              — Application files (don't modify)
  samples/          — Sample images for testing
  output/           — Detection results saved here
  SoapDetector.bat  — Double-click to run
  README.txt        — This file

REQUIREMENTS:
  Windows 10/11 (64-bit)
  No installation needed — runs out of the box

USAGE:
  1. Double-click SoapDetector.bat
  2. Choose input: Webcam, Image File, or Batch/Folder
  3. For image files, samples are in the samples/ folder
  4. Results are saved to the output/ folder
